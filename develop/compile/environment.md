# 开发环境

## 基础环境：

操作系统：MAC、Linux、Windows

中间件：Redis、Nginx

业务数据库：Mysql / Postgresql

时序数据库：TDengine

## 开发环境

GoFrame 使用的是V2.2.0版本，需要Go1.16以上版本

需要在go.mod中引用

```go
require (
    github.com/gogf/gf/v2 v2.2.0
)
```

其它[参考GoFrame的开发环境配置](https://goframe.org/pages/viewpage.action?pageId=1114389)
