# 创建设备

## 添加设备

1.登录SagooIOT物联网平台。
2.在左侧导航栏中，选择设备管理>设备，点击新增设备。
3.在新建弹框中，根据设备实际情况，按照页面提示填写信息，然后单击确定。


| 参数     | 说明                               |
| -------- | ---------------------------------- |
| 设备标识 | 设备唯一标识，在系统内全局唯一性。 |
| 设备名称 | 为设备命名，最多可输入200个字符。  |
| 所属产品 | 单选下拉框，根据实际情况选择产品。 |
| 所属部门 | 单选下拉框，选择设备归属组织。     |
